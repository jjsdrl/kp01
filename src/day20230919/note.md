1.java中的流程控制语句

```
循环结构：
	循环语句的组成部分：增量（i），循环的判断条件（决定了循环语句什么时候终止），循环体（循环执行的代码块）
	do...while...	先执行一次循环体，再判断循环条件是否成立，决定循环是否继续执行
		do{
			循环体
		}while(循环判断条件)
	while......		先判断循环条件是否成立，若成立，则执行循环体，否则结束循环
		while（循环判断条件）{
			循环体
		}
	for......	先给增量赋予初始值（只会执行一次），执行循环判断，执行循环体（若条件成立），增量自增/自减，执行循环判断条件......
		for（增量初始化值;循环判断条件;增量自增/自减）{
			循环体
		}
```
