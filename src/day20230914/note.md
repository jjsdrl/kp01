1.java中的标识符

```
标识符（取名称）：类名，方法名，变量名，包名，项目名......
类名：每个量词的首字母大写: HelloWorld Student
方法名：驼峰式命名 第一个单词的首字母小写，后面四个单词的首字母大写 getFood() getName()
变量名：驼峰式命名法 studentAge
常量名：每个字母都大写，单词与单词之间用下划线隔开 int--->Integer MAX_VALUE MIN_VALUE
包名：所有的字母都小写 com.hw.test
项目名：所有的字母小写 studentsystem student_system
```

3.在java中如何定义变量

```
变量：变量能够在jvm内存中存储数据，由变量的数据类型，变量名，变量值三部分组成

变量 ------宾馆的房间
数据类型 ------房间的类型（双人床，总统套房，大床房）
变量名 ------房间门牌号（512）
变量值 ------房间内居住的人

变量的声明以及赋值1:
	声明： int num;
	赋值：num = 100;
变量的声明以及赋值2:
	int num = 100;  String name,sex = "男";

变量名的命名规则：
	1.变量名必须由字母，数字，下划线，￥，$组成（可以使用中文，但不建议）
	2.不能以数字开头
	3.不能使用java中的关键字和保留字
	4.大小写敏感，区分大小写
建议：驼峰式命名法 见名知意
```

4.java中的运算符

```
转义符：\   \n 换行 \t 制表位
算术运算符： + - * / %
一元运算符： ++ 自增 -- 自减
比较运算符： > >= < <= == != 结果为boolean类型值（true/false）
赋值运算符： = += *= /= %=
逻辑运算符： && 与   || 或   ！非  结果为boolean类型值（true/false）
位运算符：& 与运算   | 或运算   >> 向右移位运算   << 向左移位运算   ^ 异或运算
三元运算符： 表达式（该表达式结果的类型必须为boolean类型）？值1：值2    表达式结果为true，则三元运算结果为值1，否则结果为值2
```
