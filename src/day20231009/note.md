1.java是一门**面向对象**的开发语言

```
面向过程的开发语言：c语言		注重的是数据结构和算法，程序中最小的单元是函数
面向对象的开发语言：java语言	注重的是类中的属性和方法，程序中最小的单元是类

案例：把大象放入到冰箱需要几个步骤

面向过程：
	定义三个函数：
		第一步：调用把冰箱门打开的函数
		第二步：调用把大象放入到冰箱中的函数
		第三步：调用把冰箱门关闭的函数
		
面向对象：
	定义两个类：大象类（大象走进冰箱的方法），冰箱类（打开冰箱门的方法和关闭冰箱门的方法）
		第一步：调用冰箱对象的打开冰箱门的方法
		第二步：调用大象对象的走进冰箱的方法
		第三步：调用冰箱对象的关闭冰箱门的方法
```

2.java中类和对象

```
类：类是对象的抽象	在现实生活中，把具有相同属性和行为的一类事物抽象出来，形成脑海中的一个概念，这个概念就是一个类


对象：对象是类的实例	万物皆对象，看得见摸得着的事物都是一个个的对象


老虎：类	长白山上的东北虎：类 
电脑：类	江西省九江市濂溪区鲲鹏产业学院512教室的讲台上的笔记本电脑：对象
```

3.java中的类该如何去定义

```
修饰符 class 类名{
	类体
}

类体的组成：属性，方法，构造方法，代码块，内部类......

注意：
	1.在一个以.java为后缀名的java源文件中可以定义多个类
	2.在一个以.java为后缀名的java源文件经过编译之后的字节码文件个数和该源文件中定义的类的个数相同
	3.在一个以.java为后缀名的java源文件中只能拥有一个public修饰的类，且类名必须要和源文件名称保持一致

属性：变量（全局变量，类的成员变量）

	全局变量：定义在方法的外部，作用于整个类（该类中任何地方都可以访问该变量）
            拥有默认值	可以直接访问
            int 0
            double 0.0 
            boolean false
            String null 引用数据类型都为null
            
	局部变量：定义在方法的内部，作用于整个方法（该变量在该方法内可以访问，即该方法的花括号内）
			没有默认值	在访问之前需要进行赋值（初始化）
			
方法：
	1.有参有返回值的方法		String marry(double money){}
	2.有参无返回值的方法		void shopping(double money){}
	3.无参有返回值的方法		String cooking(){}
	4.无参无返回值的方法		void school(){}
	
```
