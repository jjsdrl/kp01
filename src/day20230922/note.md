1.java中的数组

```
数组：在jvm内存中开辟的一块连续的内存空间，用来存储一组具有相同数据类型的数据

数组-----------------一组抽屉  						变量-------房间
数组名---------------抽屉名称	
元素-----------------一个抽屉
下标（从0开始）-----第几个抽屉
长度-----------一组抽屉的个数

一维数组的定义：
	int[] arr = new int[10];	动态声明数组	没有在声明数组时给数组赋值 
	int[] arr = [1,3,4];		静态声明数组	在声明时给数组赋值
	int[] arr = new int[]{1,2,3}
	
	变形：int arr[] = new int[10];		int []arr = new int[10]
	数组的访问：数组是通过下标来进行访问的
```
